A C++ program is composed of one or more functions. Each function is identified by its name. the name of the function is chosen by the programmer. The exception to this rule is that there must exist one and only one function called main in any given program.
The program starts executing from the first instruction in the function main. This function executes instruction and invokes other functions to perform their operations. The other functions may be part of the same program or be members of libraries of previously developed functions. The standard library is a collection of C++ functions designed to provide, among other things, standard I/O facilities.

6.3.4 Variable declaration

type-specifier list-of-variables;
The list-of-variables is a comma separated list of identifiers representing the program variables. Each named variable possesses as fundamental type as given by the type-specifier.
The type-specifiers for the five fundamental types are introduced by the reserved keywords :
int (integer)
long (long integer)
char (character)
float (single precision floating point)
double (double precision floating point)

Following are example of variables declarations:

The operators + and – are unary when they take only one operand, as in +x or –m*n.

6.3.10 Input and Output
The C++ library provide very convenient facilities for input and output. In C++ , input is read from and output is written to streams. When iostream.h is included in a program, several standard streams are defined automatically.
The stream Cin is used for input, which is normally read from the user's keyboard .
The stream cout is used for output, which is normally send to the user's display.
The insersion operator , << , inserts data into a stream, thus the statement
Cout<<500;
Places the value 500 in the standard output steam cout, and will be printed on the user's display.
A series of << operators can be used to output several data values with a single statement. For example,
Cout<<500<<600<<700;
Output the three numbers in the form : 500600700
The statement
Cout<<" this is a string ";
Prints the string literal this is a string
The extraction operator, >> , fetches data from a specified input stream, for example, using the predefined input stream cin, we can direct the input from the keyboard to a variable X as follows :
cin>> X;
